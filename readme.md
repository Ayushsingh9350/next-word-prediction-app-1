# Next Word Prediction Web App

This project is a Next Word Prediction model built using LSTM and deployed using Streamlit.

## 🚀 Features

* Predicts next word from input text
* Built using TensorFlow & LSTM
* Simple web interface using Streamlit

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🧠 Model

* Embedding + LSTM + Softmax
* Trained on text dataset

## 📌 Example

Input: `the king`
Output: `was`
